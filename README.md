# Emotion Detection and Music Recommendation System

## Project Description
This project detects human emotions from facial expressions using a deep learning model and recommends music based on the detected emotion.

## Technologies Used
- Python
- Deep Learning
- Convolutional Neural Network (CNN)
- OpenCV
- TensorFlow / Keras

## Dataset
The dataset contains facial expression images categorized into multiple emotions.

Dataset folders:
- train
- test

Each emotion class contains images used for training and testing the model.

## Working of the System
1. The system captures or reads a face image.
2. The CNN model analyzes the facial expression.
3. The emotion is predicted.
4. Based on the emotion, suitable music is recommended.
